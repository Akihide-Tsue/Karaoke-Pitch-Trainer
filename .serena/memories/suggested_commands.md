# Suggested Commands

## Development

- `pnpm dev` - 開発サーバー起動
- `pnpm build` - プロダクションビルド
- `pnpm start` - ビルド済みサーバー起動

## Quality Checks

- `pnpm run typecheck` - TypeScript型チェック (react-router typegen + tsc)
- `pnpm run lint` - Biome lint
- `pnpm run lint:fix` - Biome lint + 自動修正
- `pnpm run format` - Biome format

## Git (Darwin)

- `git status` / `git diff` / `git log`
- `gh pr create` / `gh pr view` - GitHub CLI

## System (Darwin)

- `ls`, `find`, `grep` - ファイル操作
- Serenaツール推奨: list_dir, find_file, search_for_pattern, get_symbols_overview, find_symbol
